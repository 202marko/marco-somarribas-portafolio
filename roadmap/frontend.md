# Roadmap – Frontend

Este roadmap define mi plan de crecimiento en el área de desarrollo frontend,
enfocado en la creación de interfaces funcionales, accesibles y orientadas a
la experiencia del usuario.

---

## Fundamentos
- HTML semántico
- CSS (Flexbox, Grid, diseño responsive)
- JavaScript (DOM, eventos, lógica básica)

---

## Diseño y UX
- Principios de usabilidad
- Mobile First
- Accesibilidad básica (a11y)

---

## Herramientas y buenas prácticas
- Control de versiones con Git
- Organización de proyectos frontend
- Documentación clara del código

---

## Próximos pasos
- Frameworks frontend
- Consumo de APIs
- Optimización de rendimiento
