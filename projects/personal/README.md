# Proyectos Personales

Esta sección contiene proyectos desarrollados de forma independiente con el objetivo
de reforzar habilidades técnicas, experimentar con nuevas tecnologías y aplicar
conocimientos en escenarios prácticos.

Los proyectos personales reflejan iniciativa, curiosidad técnica y aprendizaje continuo.

Cada proyecto incluirá:
- Objetivo del proyecto
- Tecnologías utilizadas
- Problema que resuelve
- Enlace al repositorio correspondiente
