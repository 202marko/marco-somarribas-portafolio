# Proyectos Universitarios

En esta sección se documentan los proyectos desarrollados durante la carrera de
Ingeniería en Desarrollo de Software en la Universidad CENFOTEC.

Estos proyectos tienen como objetivo aplicar conceptos teóricos, metodologías de
desarrollo, buenas prácticas de programación, análisis de requerimientos y documentación
técnica, simulando escenarios reales del entorno profesional.

---

## Áreas abordadas
- Programación (Java, estructuras de datos, POO)
- Bases de datos y modelado (SQL, diseño lógico y físico)
- Desarrollo web (HTML, CSS, JavaScript)
- Análisis y diseño de sistemas
- Ética, profesionalismo y documentación

---

## Proyectos
> *Los proyectos se irán agregando conforme avance la carrera.*

Cada proyecto incluirá:
- Objetivo del sistema
- Tecnologías utilizadas
- Rol desempeñado
- Enlace al repositorio correspondiente
